# AI Usage Policy

This project may use artificial intelligence tools to assist with development, documentation, localization, review, refactoring, testing, terminology research, and translation quality checks.

AI tools are used as assistants. They are not considered authors, maintainers, contributors, or copyright holders of this project.

## Human Review Required

All AI-assisted contributions must be reviewed, edited, and approved by a human contributor before being submitted or merged.

Contributors are responsible for the content they submit, including any AI-assisted code, documentation, translations, terminology suggestions, or localization edits.

By submitting AI-assisted material, contributors confirm that they have reviewed it and believe it is suitable for inclusion in the project.

## Code Generated with AI Assistance

AI-assisted code may be accepted if it meets the same requirements as any other code contribution.

Contributors must ensure that AI-assisted code:

* is understandable and maintainable;
* is compatible with the project license;
* does not knowingly copy code from unauthorized sources;
* does not introduce security issues;
* does not include secrets, credentials, private keys, tokens, or confidential information;
* is tested or reasonably verified before submission.

AI-generated code should not be submitted blindly.

## Localization and Translation Assisted by AI

AI tools may be used to assist with Spanish translation, localization review, terminology consistency, grammar, style, and quality assurance.

However, localization contributions must be reviewed by a human contributor before inclusion.

Contributors must ensure that AI-assisted localization content:

* reflects an intentional human localization decision;
* does not blindly reproduce machine output;
* does not include unauthorized third-party translations;
* does not misrepresent itself as official Square Enix localization;
* respects the terminology, tone, and quality standards of the project.

The project may reject AI-assisted translations that are inaccurate, inconsistent, overly literal, stylistically inappropriate, or not sufficiently reviewed.

## Prompts and Source Material

Contributors must not submit prompts or source material containing:

* secrets, API keys, access tokens, credentials, or private certificates;
* private personal data;
* confidential business information;
* proprietary source code that they are not allowed to share;
* copyrighted third-party material that they are not allowed to use;
* unreleased or leaked game content.

When using AI tools, contributors are responsible for checking the terms and data usage policies of the tools they use.

## Ownership and Licensing

AI-assisted contributions are subject to the same contribution terms as non-AI contributions.

For code contributions, this means the MIT License applies unless explicitly stated otherwise.

For localization, translation, glossary, terminology, and other linguistic contributions, the separate localization contribution terms in `CONTRIBUTING.md` apply unless explicitly stated otherwise.

By submitting AI-assisted contributions, contributors confirm that they have the necessary rights to grant the project the applicable license.

## Disclosure

Contributors are encouraged to disclose meaningful AI assistance in pull requests when it materially affected the contribution.

Examples of useful disclosure include:

* AI-assisted refactoring;
* AI-assisted translation draft reviewed by a human;
* AI-assisted terminology review;
* AI-assisted test generation;
* AI-assisted documentation rewrite.

Minor assistance, such as grammar correction, autocomplete, formatting, or small wording suggestions, does not need to be disclosed unless the contributor considers it relevant.

## Maintainer Rights

The project maintainer may request clarification about AI-assisted contributions.

The maintainer may reject, edit, rewrite, remove, or replace AI-assisted material if there are concerns about quality, licensing, originality, security, maintainability, or project direction.
